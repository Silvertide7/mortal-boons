# Mortal Boons

A NeoForge mod (Minecraft 1.21.1) that adds roguelike boons: spend experience and
items to roll a boon, each with a tier (I–IV) that scales its effect. Hold up to
three at once — but lose them all when you die.

Status: base project scaffold. Boon system not yet implemented.

## Development

Open the repository in IntelliJ IDEA. Run `./gradlew --refresh-dependencies` if the
IDE is missing libraries, and `./gradlew clean` to reset generated output.
